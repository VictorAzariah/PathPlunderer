## ūüĒć What is PathPlunderer?

**PathPlunderer** is a fast, multi-threaded Python tool for brute-forcing web directories and files. Built for penetration testers and bug bounty hunters, it combines speed and flexibility ‚ÄĒ supporting recursive scanning, custom headers, proxy routing, Basic Auth, file extension fuzzing, and more.

Think of it as a lightweight, customizable alternative to tools like `dirb` or `gobuster` ‚ÄĒ written entirely in Python.

---

## ‚ú® Features

- **Recursive scanning** ‚ÄĒ automatically dives into discovered directories
- **Multi-threaded** ‚ÄĒ configurable thread count for fast enumeration
- **Extension fuzzing** ‚ÄĒ append file extensions like `.php`, `.txt`, `.bak` to every wordlist entry
- **Auto-protocol detection** ‚ÄĒ input just a domain; PathPlunderer finds the right protocol
- **HTTP method support** ‚ÄĒ GET, POST, HEAD, PUT, OPTIONS, PATCH
- **Proxy support** ‚ÄĒ route traffic through Burp Suite or any HTTP(S) proxy
- **Custom headers & cookies** ‚ÄĒ inject session tokens, auth headers, and more
- **Basic Auth** ‚ÄĒ authenticate with username and password
- **Wildcard detection** ‚ÄĒ force wildcard processing to avoid false positives
- **Custom/random User-Agent** ‚ÄĒ evade basic WAF fingerprinting
- **Follow redirects** ‚ÄĒ optionally chase 3xx responses
- **Insecure mode** ‚ÄĒ bypass SSL/TLS certificate validation
- **Output logging** ‚ÄĒ save results to a file
- **Status code filtering** ‚ÄĒ show only the codes you care about
- **Response size display** ‚ÄĒ spot anomalies in response sizes

## ‚öôÔłŹ Usage

### Basic scan
```bash
python PathPlunderer.py -u https://example.com -w Wordlists/common.txt
```

### With threads and extension fuzzing
```bash
python PathPlunderer.py -u https://example.com -w Wordlists/common.txt -t 50 -x php,html,txt
```

### With proxy (e.g. Burp Suite)
```bash
python PathPlunderer.py -u https://example.com -w Wordlists/common.txt -p http://127.0.0.1:8080
```

### With Basic Auth
```bash
python PathPlunderer.py -u https://example.com -w Wordlists/common.txt --user admin --pass secret
```

### With custom headers and cookies
```bash
python PathPlunderer.py -u https://example.com -w Wordlists/common.txt \
  -H 'Authorization: Bearer TOKEN' \
  -c 'session=abcdef123456'
```

### Save results to file
```bash
python PathPlunderer.py -u https://example.com -w Wordlists/common.txt -o results.txt
```
